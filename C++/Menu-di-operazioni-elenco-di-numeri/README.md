darà un elenco di operazioni possibili tra cui l'operatore potrà scegliere
cosa vuole calcolare di un elenco di numeri interi tra:
1 - Media aritmetica; 
2 - Conteggio numeri primi; 
3 - Conteggio numeri pari e dispari;
